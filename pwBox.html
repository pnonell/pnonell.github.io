PwBox

Context:
    Web projects II (3rd grade)

Technologies:
    PHP, Slim framework, Javascript, HTML, CSS, SQL

Description:
    This project consisted in programming a cloud storage website (as Drive or Dropbox) with a group of 3 people using web programming languages and Slim Framework. Among the functionalities of this website were the registration and login of users, the management of documents and folders and the possibility of sharing these documents with other users.
    This allowed us to put into practice knowledge of both the Back-End for the most part and the Front-End for the least part.

Github repository: github.com/pnonell/PWBox
