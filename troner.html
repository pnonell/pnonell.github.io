Troner

Context:
    Object-oriented programming and design (2nd grade)

Technologies:
    Java, Sockets, Server-Client structure, Model-view-controller pattern

Description:
    This project consisted in programming the popular Tron game in Java using object-oriented programming with a group of 5 people. The final project was composed by a Server software, which administrates the games and the users; and a Client software which is used by the user to play 2,3 or 4 multiplayer games with different game modes.
    This project allowed me to put in practice a Server-Client software and decide how to structure the functions of it. It also helped me to understand the Model-View-controller pattern.

Github repository: github.com/pnonell/Troner
