LS Dungeon

Context:
    Web projects I (3rd grade)

Technologies:
    Javascript, CSS, HTML

Description:
    This project consited in programming a Dungeon Crawl game using front-end web programming languages (html, javascript, css) with a group of 3 people.
    The game included the functionalities of choosing an avatar, username and, of course, move the player thourgh the maze. During the game, the player could can collect objects which will help him to defeat the enemies that will block his way to the end of the game. It also included a mini-map that became visible as the player progressed. To make the game more fun, we gave it an aesthetic similar to the university intranet and included objects and enemies of the faculty.
    This project allowed me to put in practice the main front-end web languages in different ways. The game has several functionalities so we took advantage of all the features that these languages gave us.
