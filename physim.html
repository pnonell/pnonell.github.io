Physical simulation

Context:
    Physical simulation (3rd grade)

Technologies:
    Processing, Java

Description:
    This project consisted in programming simulations and games to represent physical phenomenon related to forces. There are 3 different parts: a Snake game where the snake body moves with a spring behaviour, a representation and calculus of a satellite orbiting the Earth in different ways and a representation of a bouncing ball in different situations and simulations.
    This project allowed me to understand better some physical laws and how to represent them in a visual way.

Github repository: github.com/pnonell/physical-simulation
